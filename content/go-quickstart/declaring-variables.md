Here are all the ways you can declare and assign value to variable: 
```go
var counter int
counter = 10
```
But we can do shorter
```go
var counter int = 10
```
More shorter!
```go
var counter = 10
```
And the most shortest... Notice, this one works only inside functions!
```go
counter := 10
```
